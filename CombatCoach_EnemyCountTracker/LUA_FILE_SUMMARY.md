# Lua file summaries for CombatCoach_EnemyCountTracker

Short one-line descriptions for each Lua file in this addon.

- `EnemyCountTracker.lua`: Core enemy-count tracking logic.
- `EnemyCountTracker_Debug.lua`: Debug utilities and test helpers for the tracker.
- `EnemyCountTracker_UnitFrames.lua`: Unit frame helpers and overlays.
- `EnemyCountTracker_DebuffOverlay.lua`: Debuff overlay rendering on unit frames.
