# Lua file summaries for CombatCoach_DynamicBuffTracker

Short one-line descriptions for each Lua file in this addon.

- `DynamicBuffTracker.lua`: Core dynamic buff tracking runtime.
- `DynamicBuffTracker_CDM.lua`: CDM (condition/definition/model) for dynamic buffs.
- `DynamicBuffTracker_Icons.lua`: Icon handling for buff displays.
- `DynamicBuffTracker_CombatCoach.lua`: Integration layer with CombatCoach.
- `DynamicBuffTracker_Defaults.lua`: Default tracker configurations and presets.
- `DynamicBuffTracker_Helpers.lua`: Helper utilities used across the module.
- `DynamicBuffTracker_Scan.lua`: Scanning logic for detecting buffs/debuffs.
- `DynamicBuffTracker_State.lua`: State management for dynamic buff entries.
