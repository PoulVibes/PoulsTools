# Lua file summaries for CombatCoach

Short one-line descriptions for each Lua file in this addon.

- `CombatCoach.lua`: Core addon initialization, event registration and main runtime logic.
- `CombatCoach_Menu.lua`: Menu definitions and top-level menu helpers.
- `CombatCoach_Menu_MainPanel.lua`: Main menu panel UI and its behaviors.
- `CombatCoach_Menu_SubPanel.lua`: Subpanel UI components for the menu system.
- `CombatCoach_Widgets.lua`: Shared widget definitions used across the addon.
- `CombatCoach_Widgets_Input.lua`: Input widget implementations and keyboard handlers.
- `CombatCoach_Profiles.lua`: Profile management (save/load profiles and defaults).
- `CombatCoach_Profiles_UI.lua`: UI for editing, selecting and managing profiles.
- `CombatCoach_Profiles_Menu.lua`: Profile-related menu entries and glue.
- `ExampleAddon_CombatCoach.lua`: Example / sample addon demonstrating integration points and usage.
