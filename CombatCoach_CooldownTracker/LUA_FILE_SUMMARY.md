# Lua file summaries for CombatCoach_CooldownTracker

Short one-line descriptions for each Lua file in this addon.

- `CooldownTracker.lua`: Core cooldown-tracking logic and timers.
- `CooldownTracker_State.lua`: State management for tracked cooldowns.
- `CooldownTracker_CombatCoach.lua`: CombatCoach integration layer for cooldown tracking.
- `CooldownTracker_CombatCoach_LookAndFeel.lua`: Look-and-feel and UI styling helpers.
