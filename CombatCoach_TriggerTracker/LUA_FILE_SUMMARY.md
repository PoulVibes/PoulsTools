# Lua file summaries for CombatCoach_TriggerTracker

Short one-line descriptions for each Lua file in this addon.

- `TriggerTracker.lua`: Core trigger tracker logic and runtime.
- `TriggerTracker_DB.lua`: Persistence and database helpers for triggers.
- `TriggerTracker_Icons.lua`: Icon handling for triggers; includes `TriggerTracker_ExtendTimer` for extending active buff timers.
- `TriggerTracker_Export.lua`: Export/import logic for trigger sets.
- `TriggerTracker_CombatCoach.lua`: CombatCoach integration layer for triggers.
- `TriggerTracker_CreationFrame.lua`: UI for creating triggers.
- `TriggerTracker_CreationFrame_New.lua`: New-creation flow for triggers.
- `TriggerTracker_CreationFrame_Drag.lua`: Drag handling inside creation UI.
- `TriggerTracker_CreationFrame_Flyout.lua`: Flyout widgets used in the creation UI.
- `TriggerTracker_CreationFrame_Widgets.lua`: Reusable widgets for the creation UI.
- `TriggerTracker_CreationFrame_Helpers.lua`: Helper utilities for trigger creation.
- `TriggerTracker_Defaults.lua`: Default trigger definitions and constants.
- `TriggerTracker_State.lua`: Runtime state management for triggers.
