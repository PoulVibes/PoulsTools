# Lua file summaries for CombatCoach_shmIcons

Short one-line descriptions for each Lua file in this addon/library.

- `shmIcons.lua`: Main entry and public API surface for the shmIcons library.
- `shmIcons_API.lua`: API definitions and exported helper functions.
- `shmIcons_Core.lua`: Core runtime logic and icon management.
- `shmIcons_Visuals.lua`: Rendering and visual helpers for icons.
- `shmIcons_Hotkeys.lua`: Hotkey registration and handling for icons.
- `shmIcons_EditMode.lua`: Edit-mode UI for arranging and configuring icons.
- `shmIcons_IconFrame.lua`: Implementation of the icon frame widget.
- `shmIcons_API_System.lua`: System-level API glue and platform helpers.
- `shmIcons_CombatCoach.lua`: Integration layer between shmIcons and CombatCoach.
- `shmIcons_IconRuntime.lua`: Runtime behavior and lifecycle for icon instances.
- `shmIcons_EditModeGroupFrame.lua`: Group frame UI for edit mode.
- `shmIcons_IconFrame_Behavior.lua`: Reusable behaviors for icon frames.
- `shmIcons_IconFrame_DragHelpers.lua`: Drag-and-drop helper utilities.
- `shmIcons_EditModeGroupFrame_Drag.lua`: Drag handling for grouped frames.
- `shmIcons_EditModeGroupFrame_Settings.lua`: Settings and persistence for edit-mode groups.
